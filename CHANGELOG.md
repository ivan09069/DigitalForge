## Format

### Added
New features, new files, new capabilities.

### Changed
Modifications to existing features.

### Deprecated
Features marked for future removal.

### Removed
Features removed from the forge.

### Fixed
Bug fixes and corrections.

### Forged
Initial creation of something new.

### Reforged
Significant rework of existing feature.

### Cooled
Features moved to maintenance mode.
