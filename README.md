## Core Philosophy

### 1. Visibility Before Mystery
All systems expose state, load, latency, and failure edges.  
Trust is earned through transparency.

### 2. Energy Is Directed
No decorative motion. No idle animation.  
Movement shows flow, stress, instability, or direction of attention.

### 3. Signals, Not Instructions
DigitalForge does not tell operators what to do.  
It provides predictive hints and directional signals.  
The operator decides. Skill matters.
